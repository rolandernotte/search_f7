# Search F7

Exemple d'app mobile Cordova/F7

Utilisation de l'API Foursquare
Utilisation de l'API OpenStreetMaps

# utilisation

1. Télécharger et dézipper (étape 1 dans branche Step1, étape 2 dans branche Step2,... état actuel dans branche master)
2. Dans `www/index.js`, remplacer `CLIENT_ID` et `CLIENT_SECRET` par vos identifiants Foursquare

## branche step1
Géolocalisation, appel Ajax à l'API Foursquare, liste des pizzérias les plus proches
